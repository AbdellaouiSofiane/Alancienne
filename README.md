# Alancienne

  # Ajout d'un produit:
    Permet d'ajouter un produit dans la base de données.
    Tous les champs sont obligatoires.

![image](https://user-images.githubusercontent.com/62038668/109812621-92e7c480-7c2c-11eb-8d82-a54c8ce53872.png)


  # Creation d'un panier :
    Permet de créer et de modifier son panier de manière dynamique.
    Seul les produits avec un stock restant positif sont affichés.
    la quantité maximum pour chaque produit est égale au stock restant disponible.
    Un champ représentant le total commandé en prix et en quantité sera présent en bas de la liste.
